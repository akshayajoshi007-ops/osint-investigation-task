# OSINT Investigation Report

## Objective

To perform an Open Source Intelligence (OSINT) investigation by collecting publicly available information about a target and analyzing their digital footprint.

## Tools Used

* Google Search Engine
* Social Media Platforms (LinkedIn, GitHub, Instagram)
* Have I Been Pwned (Breach Checker)

## Steps Performed

1. Selected a demo target profile for investigation.
2. Performed Google search using basic and advanced queries.
3. Identified usernames and checked their presence across platforms.
4. Analyzed social media accounts for publicly available information.
5. Checked email exposure using a breach detection tool.
6. Collected and correlated data from multiple sources.
7. Analyzed risks based on gathered information.

## Importance of OSINT

OSINT helps in gathering publicly available information for cybersecurity investigations. It is widely used in ethical hacking, digital forensics, and threat intelligence to understand potential risks and exposures.

## Outcome

Successfully collected and analyzed publicly available data of the target. Identified digital footprints, correlated information, and assessed potential security risks.

## Conclusion

The OSINT investigation was successfully completed using ethical methods. The analysis shows how easily publicly available data can be used to understand an individual's online presence. This highlights the importance of maintaining online privacy and security.
